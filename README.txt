Tetris Deluxe Android/iOS PWA Package

Upload this whole folder to an HTTPS host. Open index.html in Chrome on Android or Safari on iOS.

Android Chrome:
1. Open the hosted URL.
2. Tap the three-dot menu.
3. Choose Add to Home screen or Install app.

iOS Safari:
1. Open the hosted URL in Safari.
2. Tap Share.
3. Choose Add to Home Screen.

Note: local file:// HTML will not install properly as a PWA on iOS or Android. HTTPS hosting is needed for best app-like behavior and offline caching.
